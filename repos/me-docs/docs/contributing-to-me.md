# Contributing to Meta Earth

> Meta Earth Devs, as an open-source technology community, is jointly maintained and built by community contributors from around the world. Regardless of your field or professional level, there are many ways to contribute to Meta Earth and make a real difference to the entire Meta Earth community.

Listed here are known ways to participate in Meta Earth work and how you can get started. Even if what you want to do is not listed here, we welcome contributions. If you're still not quite sure, please actively submit an Issue on [Github](https://github.com/openmetaearth/) or post on the Forum seeking community advice.

---

## 🔧 Code Programming

Meta Earth, as a modular, high-performance, infinitely scalable multi-dimensional fusion blockchain underlying technology public chain, is a very large project that cannot be completed by just a few core developers. We very much hope that more community developers can write and maintain code together.

**How to Start:**

- We use the **"Help Wanted"** label on features or bug fixes where everyone can help - this is the best entry point for you to start contributing code
- Check out our repository: [https://github.com/openmetaearth/](https://github.com/openmetaearth/)
- The Meta Earth community will regularly release related activities that you can follow and participate in on the forum to solve various technical problems

---

## 📝 Writing Documentation

With the emergence of large amounts of code, products, tools, and applications, a large amount of concise and clear documentation is needed to help users understand, learn, and use them.

**We Need You To:**

- Help write various excellent documents required for each project (including video tutorials)
- Translate documents into your local language

📚 This Meta Earth documentation repository is still being continuously revised, and we welcome contributions: [https://github.com/openmetaearth/me-docs](https://github.com/openmetaearth/me-docs)

## 🚀 Product Introduction

| Product | Description |
|---------|-------------|
| **Blockchain Explorer** | The blockchain explorer is your gateway to accessing Meta Earth network data. You can use it to browse real-time data on blocks, transactions, accounts, nodes, and other on-chain activities. |
| **Plugin Wallet** | Google browser plugin wallet, designed to make MEC digital asset management as seamless as breathing and as secure as a fortress. |
| **ME PASS** | Meta Earth's official ecosystem entry application, empowering users to build digital sovereign identities, conveniently manage on-chain identities, and participate in ecosystem interactions. Users can create MEC addresses, apply for, and manage ME IDs in ME Pass. |
| **ME Academy** | Meta Earth's educational platform that helps users find their own path in the Web3 era by providing modular courses and authoritative blockchain guides. |
| **Cross-Chain Bridge** | Adopts a hybrid architecture design of "distributed relayer + smart contract adaptation," integrating core technologies such as distributed consensus and smart contract adaptation, supporting cross-chain interaction between mainstream public chains like BSC, TRON, and Meta Earth. |
