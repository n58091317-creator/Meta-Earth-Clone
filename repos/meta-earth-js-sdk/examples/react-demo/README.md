# Meta Earth JS SDK React Demo

A demo application for Meta Earth JS SDK built with React, TypeScript, and Ant Design.

## Highlights

- **Modern UI** - Built with Ant Design 5.x components
- **Modular Architecture** - Functional panels are organized independently
- **Live Activity Log** - Clear, timestamped operation logs
- **Responsive Layout** - Adapts to different screen sizes
- **Full Feature Coverage** - Demonstrates all SDK capabilities

## Usage Guide

### 1. Initialize the SDK

- Click the `Initialize SDK` button
- Wait until the log shows `SDK initialized successfully`

### 2. Wallet Management

- **Create Wallet**: Generate a new mnemonic and address
- **Import Wallet**: Import with either a mnemonic or a private key
- **Address Conversion**: Convert between `0x` and ME addresses
- **View Addresses**: Display the current wallet list

### 3. Queries

- **Balances**: Check balances on the HUB or Rollapp layer
- **Network**: View network status and node version
- **Transactions**: Fetch transaction details by hash
- **Governance**: View active proposals

### 4. Transfer

- Enter the sender and recipient addresses
- Input the amount and denom
- Select the layer (`HUB` or `Rollapp`)
- Add an optional memo
- Click `Send Transfer`

### 5. Staking

- **Stake**: Stake tokens with flexible duration
- **Unstake**: Withdraw staked tokens
- **Claim Rewards**: Claim rewards generated by staking

### 6. Governance

- **Submit Proposal**: Create a text or software upgrade proposal
- **Vote**: Vote on a proposal (yes / no / veto / abstain)
- **View Proposals**: Query currently active proposals

### 7. WASM Contracts

- **Store Code**: Upload WASM contract code to the chain
- **Instantiate**: Create a contract instance from a Code ID
- **Execute**: Call contract functions

## Activity Log

- Every action writes a detailed log entry
- Log entries include timestamps for easier debugging
- Logs can be cleared manually
- The log panel stays at the bottom of the page

## Component Overview

### SDKStatus

Displays core SDK runtime information:

- SDK version
- Runtime environment (`browser` / `node`)
- Initialization status
- Current network

### LogPanel

Displays real-time logs for all operations:

- Auto-scrolls to the latest entry
- Timestamped log items
- Supports clearing the log list

### WalletManager

Wallet-related actions:

- Create a new wallet from a mnemonic
- Import a wallet from mnemonic / private key
- Convert address formats
- Display wallet addresses

### QueryPanel

General lookup tools:

- Balance query
- Network information
- Transaction details
- Active proposals

### TransactionPanel

Transfer operations:

- Sender address
- Recipient address
- Transfer amount
- Denom
- Layer selector
- Memo

### StakingPanel

Staking-related actions:

- Stake tokens
- Unstake tokens
- Claim rewards

### GovernancePanel

Governance features:

- Submit proposals
- Vote on proposals
- View proposals

### ContractPanel

Contract operations:

- Store code
- Instantiate contracts
- Execute contracts
- Query contract state

## Development Notes

### Add a New Feature

1. Create a new component in `src/components/`
2. Use the SDK instance inside the component
3. Record operation logs with `addLog`
4. Build the UI with Ant Design components

### Customize Styles

Edit `src/App.css` to add custom styles.

### Extend the SDK

If you want to test a new SDK capability:

1. Add new form fields to the component
2. Call the new SDK method
3. Add log output
4. Handle error messages

## Notes

### Test Environment

- The demo currently uses the test network
- Do not send real tokens
- Generated addresses are for testing only
